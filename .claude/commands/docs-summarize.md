Analyze the codebase and update the codebase summary.

## Process

Use Agent tool (model: sonnet) to launch a **Docs Manager** agent:

1. Analyze the current codebase structure and recent changes
2. Update `./docs/codebase-summary.md` with current state
3. Ensure the summary reflects the latest code, architecture, and dependencies

Use `./docs/` directory as the source of truth for documentation.

**IMPORTANT:** Do NOT implement code. Only update documentation.
