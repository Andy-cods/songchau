## Workflow

1. Use Agent tool (model: opus) to launch a **UI/UX Designer** agent to create a design plan with TODO tasks in `./plans/`
2. Implement the design step by step
3. If not specified, create in pure HTML/CSS/JS
4. Report to user with a summary, ask for review and approval
5. If approved, update `./docs/design-guidelines.md` if needed

## Design Principles
- Clean, modern, responsive design
- Mobile-first approach
- Accessible (WCAG 2.1 AA)
- Consistent with existing design system
