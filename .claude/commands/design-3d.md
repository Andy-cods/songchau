## Workflow

1. Use Agent tool (model: opus) to launch a **UI/UX Designer** agent AND **Researcher** agent to create a 3D design plan in `./plans/`
2. Implement the plan step by step
3. Create immersive 3D experiences using Three.js with particle effects, custom shaders, interactive elements
4. Report to user, ask for review and approval
5. If approved, update `./docs/design-guidelines.md`

## 3D Design Requirements
- Three.js scenes with proper optimization
- Custom GLSL shaders for unique visual effects
- GPU-accelerated particle systems
- Immersive camera controls and cinematic effects
- Post-processing effects and render pipelines
- Responsive across all devices
- Optimized for real-time 60fps rendering
- Interactive elements and smooth animations

## Stack
- Three.js for 3D rendering
- GLSL for custom vertex and fragment shaders
- HTML/CSS/JS for UI integration
- WebGL for GPU-accelerated graphics
