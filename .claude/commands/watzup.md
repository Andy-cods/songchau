Review recent changes and provide a detailed status update.

## Process

1. Check `git log` for the most recent commits on the current branch
2. Run `git diff` to see any uncommitted changes
3. Run `git status` to check working tree state
4. Read `docs/PROGRESS.md` if it exists

## Report

Provide a concise summary:
- **Branch**: Current branch name
- **Recent commits**: Last 5-10 commits with descriptions
- **Uncommitted changes**: What's modified but not committed
- **Overall status**: What was done, what's in progress, what's next

**IMPORTANT:** Do NOT start implementing anything. Only report status.
